# TinyTimer

TinyTimer is a small app which will help you to manage your time. Place it anywhere on your screen and take your time!<br>
There are wide customization options available in the app.

**Usage:**<br>
RMB to open/close settings and the control panel.

**System requirements:**<br>
- Windows 7+
- .NET Framework 4.8+

*(Note: The program is currently in beta and may be unstable.)*
